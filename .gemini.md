# Day To Us - AI Assistant Context

> Quick reference for AI assistants working on this project.

## Project Overview

**Day To Us** - 커플을 위한 생활 공유 PWA 서비스

### 도메인 지식

**핵심 컨셉**: 커플이 일상을 함께 관리하고 공유하는 플랫폼

**1차 기능 (MVP)**: 공동 가계부
- 커플이 함께 사용하는 가계부
- 수입/지출 기록 및 실시간 동기화
- 카테고리별 지출 분석
- 월별 예산 설정 및 추적

**향후 확장**: 일정 공유, 사진 앨범, 기념일 관리, 위시리스트

### Tech Stack
- React 19.2.0 + TypeScript 5.9.3
- Vite 7.3.1 (build tool)
- vite-plugin-pwa 1.2.0 (PWA support)
- ESLint 9.39.1
- **axios** 1.7.9 (HTTP client)
- **date-fns** 4.1.0 (Date utility)

## Backend API

**Base URL**: http://localhost:8080

### Endpoints
- `GET /shared-spaces` - 공간 목록
- `POST /shared-spaces` - 공간 생성
- `POST /shared-spaces/join` - 공간 참여 (초대 코드)
- `GET /shared-spaces/{spaceId}/budget-entries` - 가계부 목록
- `POST /shared-spaces/{spaceId}/budget-entries` - 항목 생성
- `PUT /shared-spaces/{spaceId}/budget-entries/{entryId}` - 항목 수정
- `DELETE /shared-spaces/{spaceId}/budget-entries/{entryId}` - 항목 삭제

## TypeScript Best Practices

### File Extensions
- **`.tsx`**: React components
- **`.ts`**: Utils, types, non-React code
- **`.d.ts`**: Type declarations

### Type Definitions
```typescript
// Always type component props
interface Props {
  title: string
  onClick: () => void
}

// Use type imports
import type { FC } from 'react'
```

### Path Aliases
- `@/` → `./src/`
- Example: `import apiClient from '@/shared/api/client'`

## React Best Practices

### Component Structure
```typescript
import { useState } from 'react'
import type { FC } from 'react'

interface Props {
  title: string
}

export const Component: FC<Props> = ({ title }) => {
  const [state, setState] = useState('')
  
  return <div>{title}</div>
}
```

### Naming Conventions
- **Components**: PascalCase (`BudgetCard.tsx`)
- **Hooks**: camelCase with `use` prefix (`useBudget.ts`)
- **Utils**: camelCase (`formatCurrency.ts`)
- **Types**: PascalCase (`budget.types.ts`)

## PWA Configuration

### Service Worker
- **Strategy**: generateSW (auto-generated)
- **Update**: autoUpdate
- **Dev mode**: Enabled

### Caching
- Static assets: Precached
- Images: CacheFirst (30 days)
