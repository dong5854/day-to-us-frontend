# Day To Us - AI Assistant Context

> Quick reference for AI assistants working on this project.

## Project Overview

**Day To Us** - 커플을 위한 생활 공유 PWA 서비스

### 도메인 지식

**핵심 컨셉**: 커플이 일상을 함께 관리하고 공유하는 플랫폼

**핵심 기능 (Core)**:
- **공동 가계부**: 수입/지출 기록, 실시간 동기화, 카테고리별 분석, 월별 예산
- **일정 관리**: 커플 일정 공유, 캘린더 뷰, 카테고리별 일정
- **고정 지출**: 정기 지출 항목 관리 및 자동 반영
- **공간 관리**: 커플 전용 프라이빗 공간 생성 및 참여

**향후 확장**: 사진 앨범, 기념일 관리, 위시리스트

### Tech Stack
- **React** ^19.2.0 + **TypeScript** ~5.9.3
- **Vite** ^7.2.4 (build tool)
- **Tailwind CSS** ^4.1.18 (Styling)
- **React Router** ^7.12.0 (Routing)
- **vite-plugin-pwa** ^1.2.0 (PWA support)
- **ESLint** ^9.39.1
- **axios** ^1.13.2 (HTTP client)
- **date-fns** ^4.1.0 (Date utility)

## Backend API

**Base URL**: http://localhost:8080

### Endpoints (Examples)
- `GET /shared-spaces` - 공간 목록
- `GET /shared-spaces/{spaceId}/budget-entries` - 가계부 목록
- `GET /shared-spaces/{spaceId}/schedules` - 일정 목록
- `GET /shared-spaces/{spaceId}/fixed-expenses` - 고정 지출 목록

## TypeScript Best Practices

### File Extensions
- **`.tsx`**: React components
- **`.ts`**: Utils, types, non-React code
- **`.d.ts`**: Type declarations

### Type Definitions
```typescript
// Always type component props
interface Props {
  title: string
  onClick: () => void
}

// Use type imports
import type { FC } from 'react'
```

### Path Aliases
- `@/` → `./src/`
- Example: `import apiClient from '@/shared/api/client'`

## React Best Practices

### Component Structure
```typescript
import { useState } from 'react'
import type { FC } from 'react'

interface Props {
  title: string
}

export const Component: FC<Props> = ({ title }) => {
  const [state, setState] = useState('')
  
  return <div className="p-4">{title}</div>
}
```

### Naming Conventions
- **Components**: PascalCase (`BudgetCard.tsx`)
- **Hooks**: camelCase with `use` prefix (`useBudget.ts`)
- **Utils**: camelCase (`formatCurrency.ts`)
- **Types**: PascalCase (`budget.types.ts`)
